IframeReset
====================

Use an iframe to reset styling.

Install
-------
js min stable
```
wget -N https://raw.github.com/daxxog/iframe-reset/master/IframeReset.min.js
```

npm stable
```bash
npm install IframeReset
```
npm edge
```bash
npm install https://github.com/daxxog/IframeReset/tarball/master
```
